<b>Project: Train Ticket Reservation</b><br>
<b>Objective of Project</b><br>
The main objective of the project is to help the user to experience an online train ticket reservation portal.<br>
The user has to select various preferences from the given choices. Backend CSV files are used to retrieve and append data and text files are created which contains the reservation data.<br>
The purpose of this project is to build an application program to reduce the manual work for managing the Booking.<br>
<b>Algorithm</b><br>
The program starts by giving the option to signup or login, if the user already has a username or password, he/she can login else they would have to signup and login again.<br>
The program shall display the list of trains in which the user will get the train number, train name, source, destination, AC fare, and sleeper fare.<br>
Function to edit the schedule of trains including the train number, train name, arrival time, departure time, and source and destination function to display fare of the trains.<br>
To book a new train reservation, the user will have to enter the train number, the number of reservations, the passenger details such as phone number, age, gender and category, as well as the class which is AC or sleeper. <br>
The total fare the PNR will be printed altogether function to generate PNR number function to display the reservations made function to search passenger details using PNR number.<br>
The main section will include the options to display train list, display train schedule, display train fare display all the PNR tickets display the specific train ticket 
<br>using the PNR booking of tickets, booking of new ticket reservation, customer support quit option as well as Wi Fi.<br>
Lastly, the user gets the option to quit,if he/she wishes to do so
